# Changelog

All notable changes to this project are documented in this file.

## [Alpha-1.1.0] - 2026-04-30

### Added
- First public release.
- SVG to PNG conversion flow.
- Recursive inlining for imported SVG assets.
- Swing `ImageIcon` generation from SVG.
- ICO generation for Windows icon workflow.
- Windows EXE icon patch support via `rcedit`.
- Auto-detection for `rcedit-x64.exe`, `rcedit-x86.exe`, and `rcedit.exe`.
